Вычисление расстояния между двумя точками:

Этот алгоритм использует формулу расстояния между двумя 
точками в декартовой системе координат (теорема Пифагора).


Как работает:

На вход получает две точки в виде пар координат (x, y).
Использует формулу sqrt((x2 - x1)^2 + (y2 - y1)^2) для вычисления расстояния.
Возвращает результат.


код:
    import math

    def distance_between_points(point1, point2):
        return math.sqrt((point2[0] - point1[0])**2 + (point2[1] - point1[1])**2)

    point_a = (1, 2)
    point_b = (4, 6)
    distance = distance_between_points(point_a, point_b)
    print("Расстояние между точками:", distance)
