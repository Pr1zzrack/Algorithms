Сортировка вставками (Insertion Sort):

Постепенно строит отсортированный массив, проходя по 
списку и вставляя каждый элемент в подходящую позицию.

код:
    def insertion_sort(arr):
        for i in range(1, len(arr)):
            key = arr[i]
            j = i - 1
            while j >= 0 and key < arr[j]:
                arr[j + 1] = arr[j]
                j -= 1
            arr[j + 1] = key

    # Пример использования
    arr = [64, 34, 25, 12, 22, 11, 90]
    insertion_sort(arr)
    print("Отсортированный массив:", arr)


Объяснение:

def insertion_sort(arr):: Объявление функции сортировки вставками.
for i in range(1, len(arr)):: Внешний цикл, начиная с второго элемента.
key = arr[i]: Текущий элемент, который нужно вставить в отсортированную часть.
j = i - 1: Индекс последнего элемента в отсортированной части.
while j >= 0 and key < arr[j]:: Перемещение элементов для освобождения места для вставки.
arr[j + 1] = key: Вставка элемента в правильную позицию.
